多维潜力指标设计
针对涨停破板股次日的 短线潜力，我们选择以下多维度指标进行量化衡量：


次日开盘收益（Open Return）：次日开盘价相对前日收盘价的涨跌幅百分比。该指标体现市场对炸板的第一反应：高开表示资金情绪偏积极，低开则显示承接不足或悲观预期。例如，尾盘炸板往往导致次日低开caifuhao.eastmoney.com（资金出货迹象），而强势洗盘的炸板股次日可能平开高走或直接高开caifuhao.eastmoney.com。


次日盘中最高收益（High Return）：次日盘中最高价相对前日收盘价的涨跌幅。这一指标代表了次日理论上的最大潜在盈利。高值意味着次日盘中出现冲高行情，可提供套利空间；低值（甚至为负）则表示次日全天乏力，连前日收盘价都未超越。


次日收盘收益（Close Return）：次日收盘价相对前日收盘价的涨跌幅。该指标反映了破板股次日最终表现，是综合多方博弈后的结果。收盘收益为正说明次日股价企稳或反弹，收盘收益为负则表示延续走弱。


次日振幅（Amplitude）：用次日最高涨幅减去最低涨幅计算（即 High Return – Low Return）。该指标表示次日股价波动幅度。振幅越大，说明次日行情剧烈波动，短线机会和风险并存；振幅小则表示走势平淡。振幅结合开盘/收盘收益，可区分“高波动冲高回落”（高振幅但收涨幅有限）和“趋势单边”（低振幅、单边上行或下跌）的情形。


回封情况（Intraday Re-seal）：利用分钟数据判断当日炸板后是否曾回封涨停以及回封次数、持续时间等。例如，记录指标：“炸板后是否再次封板（是/否）”、“首次炸板时间”和“最终收盘前回封次数/持续时长”。这些由分钟数据推测的特征体现盘中多空拉锯程度：若炸板后迅速回封且保持一段时间，说明有资金反复承接；反之，一旦炸板后再未封板，则表明买盘乏力。可以进一步统计回封成功率，即在历史所有炸板案例中，当日曾经回封的比例，用于衡量整体环境下炸板股的回流意愿。


涨停维持时长：通过分钟数据计算涨停板保持的时间（例如从涨停到首次开板的间隔）。维持时间长短反映封板强度：封板维持越久说明抛压越小、主力控盘较强；过早开板则封板强度弱。此指标有助于区分“烂板”现象（多次开板回封，封板不稳）和瞬间触板就回落的弱势情形caifuhao.eastmoney.com。


板块联动性：衡量当日所属板块题材的整体热度。可统计同板块涨停股数量及其变化：例如该股所属行业或概念板块当日共有多少只个股涨停，相比前一日增加或减少多少。这反映板块行情的强弱和持续性。如果同板块集体高潮（涨停潮），则个股获得资金关注度更高，次日承接可能较好；若板块热度骤降，孤立炸板股的接力情绪会较差。板块联动性指标可以定义为：板块内涨停股数以及与前一日相比的涨停数增减，用于量化板块的情绪同步。


以上指标从多个角度刻画了炸板股次日表现的空间和特征。在数据层面，由于日线数据中的收益字段已对齐为“T日信号对应T+1日收益”，我们可以直接使用这些指标评估每次炸板事件的次日表现潜力。
潜力评分与分类体系
为了从上述多维指标中提炼出对交易有用的信息，我们可以构建潜力评分或对炸板个股进行模式分类：


潜力评分（综合打分）：将多个指标按一定权重合成为单一评分，用以量化破板次日的预期收益/风险平衡。比如，可根据历史数据训练模型，预测某炸板信号的次日收益期望，或者人为制定公式：潜力评分 = 0.5×开盘预期收益 + 0.3×盘中潜在收益 - 0.2×次日振幅风险（仅举例）。评分高的事件表示次日可能有较高溢价且风险可控。需要注意，评分模型的建立需避免使用未来数据，在实际应用中应基于炸板当日可获得的特征进行预测。


类型聚类分类：通过对历史炸板案例在次日的表现进行聚类或规则分类，识别出不同走势类型，方便针对性策略制定。常见的次日走势类型包括：


高开高走型：次日高开且一路走高，盘中基本维持强势，收盘继续上涨。此类通常对应于主力洗盘完成、承接踊跃的情况caifuhao.eastmoney.com。指标特征：开盘收益为正且较高，收盘收益为正，盘中高点接近收盘价（无明显回落）。


冲高回落型：次日可能平开或小幅高开，早盘冲高后出现明显回落，收盘涨幅收窄甚至转负。指标特征：开盘/早盘涨幅较可观，盘中最高收益远高于收盘收益，振幅大，尾盘走弱。此类情形显示抛压重，资金借反弹出货，属于短暂冲高后的回落走势。


低开反包型：次日低开，但随即出现强劲反弹走高，盘中由跌转涨，甚至收盘翻红。指标特征：开盘收益为负，随后盘中最高收益转正，收盘收益为正（或接近前收）。这表明尽管次日情绪低迷开局，但有增量资金逢低介入将股价“反包”昨日阴线，属于转弱为强的走势。


低开低走型：次日低开后继续走低，盘中几无像样反弹，收盘大幅收低。这通常是出货型炸板的表现caifuhao.eastmoney.com。指标特征：开盘大幅低开，盘中最高价仍低于昨日收盘（最高收益为负值），收盘进一步下跌。这类股票次日承接极差，延续跌势。

筛选特征与过滤条件

换手率与量能放大倍数：观察炸板当日的成交情况。当日换手率过高（如超过15%-20%）往往意味着筹码大量换手，可能是主力借机出货迹象而非洗盘caifuhao.eastmoney.com。特别是尾盘炸板时放巨量的情况，大概率属于高位出货，次日倾向低开续弱caifuhao.eastmoney.com。相反，低换手、缩量炸板更可能是洗盘行为caifuhao.eastmoney.com。因此可设定条件：剔除换手率异常放大且尾盘放量炸板的个股，优先关注放量不显著或换手适中（例如换手<10%）且缩量回调的炸板股。此外，“量能放大倍数”指当天成交额相对前几日平均的放大倍数，该值过高意味着异常成交，需警惕风险；适度放量则可能只是正常的放量换手。

封板质量：尽管炸板说明未封住，但封板过程中的质量仍可评估。例如首次封板时间（越早说明强势）、封单大小（封板时挂单金额），开板次数等。一般来说，早盘即涨停但多次开板的“烂板”表明分歧大，次日接力意愿可能较差；而临近收盘才第一次炸板的情况，若换手不大，次日有望在情绪缓和后修复。封单额偏小（例如封单不足几千万）且多次被撕开，则说明封板意愿弱，需谨慎。筛选时可剔除那些封板强度很弱（频繁开板、封单极低）的股票，重点关注封板时间较长且仅临尾盘惜败炸板的情形。

交易执行假设：模拟在信号日尾盘买入该股票，然后根据不同策略在次日卖出。主要测试三种卖出策略：


次日开盘卖出：假设次日以开盘价卖出。对应收益即open_return（百分比）。此策略侧重锁定次日开盘的溢价或避免低开风险。

次日收盘卖出：假设次日持仓至收盘再卖出。对应收益close_return。该策略反映了一整天持有的结果，受盘中波动影响，但不需要主观择时。


所有策略均以信号日收盘价为买入基准（忽略交易成本和滑点假设）。如有需要，也可以测试止损策略（例如次日若低开超过-5%则止损等），但本方案中以固定的卖出时点策略为主。


回测统计：对历史上每个炸板信号，按照上述三种策略计算单次交易收益。汇总所有交易结果，计算关键绩效指标：

最大回撤：模拟连续交易的权益曲线以评估策略风险。由于有时信号可能密集或连续，可以假设每次仅选取一只标的交易或者所有符合条件的标的等权参与。最大回撤衡量策略净值从高点回落的最大幅度，是重要风险指标。


分组统计：进一步地，将交易按照板块、市场阶段、连板高度等分组统计表现。例如，按所属行业板块汇总计算各板块内的平均收益和胜率，筛查哪类题材的炸板股机会更大；按行情周期（牛市/震荡/熊市或高低情绪阶段）分段统计策略表现，观察环境因素影响；按首板/连板分组比较，验证低位炸板和高位炸板的次日收益差异等等。

class BacktestEngine:
    def __init__(self, data, strategy, portfolio, execution):
        self.data = data
        self.strategy = strategy
        self.portfolio = portfolio
        self.execution = execution

    def run(self):
        signals = self.strategy.generate_signals(self.data)

        for t in range(len(self.data)):
            price = self.data.iloc[t]["close"]
            target_pos = signals.iloc[t]

            trade_value = abs(target_pos - self.portfolio.position) * price
            cost = self.execution.get_trade_cost(trade_value)

            self.portfolio.update(price, target_pos, cost)
